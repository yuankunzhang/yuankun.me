别忘了定期执行 brew cleanup 哦

Sep 22, 2017

使用 Homebrew 进行包管理的朋友们注意了，Homebrew 不会自动移除旧版本的软件包，你需要手动执行 brew cleanup 命令才可以。

如果你从来没调用过这个命令，那么它帮你释放出来的磁盘空间可能会吓到你。可以看看 Stackoverflow 上的这个回答，大家在调用这个命令之后纷纷表示惊呆了，有个人甚至释放掉了 17.2GB 的磙盘占用。
